How to Find Winget Package IDs

Method 1 - use PowerShell

  1. Open PowerShell or Command Prompt
  2. Type the search command with your app name:
PS C:\> winget search "google chrome"
  1. Look for the Id column in the results
  2. Copy the exact ID (e.g., Google.Chrome)
Example: Searching for "chrome" will show Google.Chrome in the Id column

Method 2 - use winget.run

  1. Go to winget.run
  2. Search for your application
  3. Click on the package you want
  4. Copy the package ID from the page
